#### GKE, 직접구성 클러스터
```
GKE를 사용하면 클러스터에 관한 설정을 다해준다 그래서 매우편리하지만 세세한 설정이 불가능하다
반면 VM 이나 인스턴스를 사용해 마스터랑, 워커 노드를 구성하고 클러스터로 구성하는 것은 설정이 번거롭지만 디테일 하다
GKE 맛보니까 로드밸런싱 하는거 금방 가능하다는게 너무 편리한 것 같다
```
