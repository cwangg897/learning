#### 알게된 점
```
js공부하면서 느낀것 메소드와 함수는 다른 개념이다 메소드는 객체지향의 개념이고 함수는 
메서드 (Method):
메서드는 객체 지향 프로그래밍에서 사용되며 특정 객체의 멤버로 존재합니다.
메서드는 객체의 상태를 조작하거나 객체에 대한 특정 동작을 수행하는 데 사용됩니다.
메서드는 해당 객체의 속성 및 상태에 액세스할 수 있으며 this 키워드를 사용하여 객체 자체에 접근합니다.
클래스 내에서 정의된 함수는 클래스의 메서드로 간주됩니다.

함수와 메서드는 개념적으로 다르지만 실제로 JavaScript에서 함수는 메서드로 사용될 수 있습니다. 함수와 메서드의 주요 차이는 함수가 독립적으로 존재하는 반면, 메서드는 객체와 관련이 있으며 객체 내에서 호출되는 것이 일반적이라는 것입니다.
```
