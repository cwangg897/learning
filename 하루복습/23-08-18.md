#### Elastic BeanStalk
```
Heroku는 node파일 올리면 배포까지 해준다 이와 비슷한게 EB이다
그래서 배포, 모니터링, 스케일 아웃 인 까지 다 제공해준다
즉 코드변경없이 코드만 업로드하면 된다 롤백도 가능

애플리케이션 배포할려면 설치하고 업데이트하고 실수하면 롤백해야하고 모니터링해야하는데
개발자는 코드만 작성하고싶다면 이런거를 해주는게 AWS Elastic Beanstalk이다

애플리케이션은 여러환경이 모아진것
그래서 여러개가 한번에 돌아가는게 바로 Beanstalk
```

#### AWS Direct Connect
```
AWS Direct Connect는 인터넷이 아닌 다른 방법으로 AWS에 연결할 수 있는 네트워킹 서비스입니다.
AWS Direct Connect를 사용하면 이전에는 인터넷을 통해 전송해야 했던 데이터를 
이제는 고객 시설과 AWS 간의 프라이빗 네트워크 연결을 통해 전달할 수 있습니다.
프라이빗 네트워크가 핵심 인터넷 안사용하고 고객 시설에서 접근하고 싶다면 이거를 이용해서 하는것이다
```


#### 인스턴스 유형
```
컴퓨팅 최적화 : 배치작업
메모리 최적화 : 메모리에서 대규모 데이터셋을 처리하는 유형의 작업에 빠른 성능을 제공합니다. 사용사례는 대규모 비정형 데이터 처릴
스토리지 최적화 : 저장소 OLTP(트랜잭션 저장) 혹은 인메모리 DB
```
